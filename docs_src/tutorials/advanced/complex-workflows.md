# 复杂工作流

> **目标**：学习如何构建复杂的多阶段流式工作流

## 概述

SAGE 支持构建复杂的数据流拓扑，包括多分支、流连接、迭代处理等高级模式。

## 内容规划

本页面正在完善中，将包含以下内容：

### 多分支 Pipeline

- 数据流的分支处理
- 按条件路由到不同分支
- 分支结果的合并

### 流连接（Join）

- 多数据流的关联处理
- Key-based Join
- 时间窗口 Join

### 迭代处理

- 循环处理直到满足条件
- 反馈循环
- 收敛判断

### 窗口和聚合

- 时间窗口
- 滑动窗口
- 会话窗口
- 窗口聚合操作
