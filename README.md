# SAGE-Pub

本仓库用于发布和维护 [SAGE](https://github.com/intellistream/SAGE) 系统的对外文档。

SAGE 是一个模块化、高性能的推理框架，支持向量增强、流数据、神经记忆与复杂数据处理。

---

## 📘 文档入口

👉 [在线文档地址](https://intellistream.github.io/SAGE-Pub/)

---

## 🧠 包含内容

- 快速开始（安装、运行示例）
- 架构图与核心模块说明
- Dashboard 使用指南
- 系统 API 文档（公开部分）
- 研究论文与引用方式
